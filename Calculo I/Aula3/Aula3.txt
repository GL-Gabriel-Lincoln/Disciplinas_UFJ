1) Dado que f e g estão definidas por f(x)=raix(x) e g(x)= x²-1 determine:

a) fof b) gog c) fog d) gof

encontrar também o dominio da função composta em cada parte.

Vamos calcular as funções compostas e seus domínios:

a) f(f(x)) = f(raiz(x)) = raiz(raiz(x)) O domínio de f(f(x)) é x >= 0, pois a raiz quadrada só é definida para números não negativos.

b) g(g(x)) = g(x²-1) = (x²-1)²-1 = x^4 - 2x² + 1 - 1 = x^4 - 2x² O domínio de g(g(x)) é todos os números reais, pois qualquer número real pode ser elevado ao quadrado.

c) f(g(x)) = f(x²-1) = raiz(x²-1) O domínio de f(g(x)) é x²-1 >= 0, ou seja, x >= 1 ou x <= -1, pois a raiz quadrada só é definida para números não negativos.

d) g(f(x)) = g(raiz(x)) = (raiz(x))² - 1 = x - 1 O domínio de g(f(x)) é x >= 0, pois a raiz quadrada só é definida para números não negativos.
